# 概念图需求简报

## 1. 使用方式

本文件用于给 AI 出图或外部美术做首轮概念图时快速对齐需求。所有新系统与新 UI 在进入实现前，优先从这里选一个简报模板扩展。

如需直接进入首批执行提示词，请优先参考：

- `docs/concept_art_prompts.md`

## 2. 战斗 HUD 概念图

### 目标

为《苍纹牌阵》绘制正式战斗 HUD 概念图，表现“剧情驱动角色战棋”的信息层级与气质。

### 必须表达

- 顶部回合与目标信息
- 右侧角色/敌情/行动面板
- 底部精简技能栏
- 左上教学卡
- 棋盘核心可读区域

### 风格锚点

- 日式油画幻想 UI
- 清爽、仪式感、可读
- 局部金饰 + 纸本面板
- 棋盘信息层允许轻玻璃语汇

### 不可偏离

- 不要厚重中世纪黑金框
- 不要像卡牌游戏主界面
- 不要让棋盘被 UI 挤压得只剩狭小中央区域

## 3. 2D 正交棋盘 / 伪 2.5D 表现概念图

### 目标

在不改变正交格规则的前提下，给出“2D 正交棋盘 + 伪 2.5D 空间感”的战棋表现方向。

### 必须表达

- 高低差
- 高台、治疗格、危险格
- 角色站位层次
- 轻微空间压迫感
- 镜头仍需清晰读取格子逻辑

### 风格锚点

- 2D 战棋可实现的空间感
- 日式油画背景
- 清亮圣庭/回廊氛围

### 不可偏离

- 不要依赖真实 3D 透视
- 不要把规则层做成真正难读的斜坐标
- 不要过暗
- 不要地形细节压过格子阅读

## 4. 章节地图 / 营地 / 结算概念图

### 目标

统一章节流程 UI 的正式视觉语言。

### 必须表达

- 章节节点推进感
- 营地角色卡与整备面板
- 结算后的奖励与章节总结
- 与战斗 HUD 一致的世界观语汇

### 风格锚点

- 战役管理感
- 纸本 + 金饰 + 柔和笔触
- 不失操作清晰度

### 不可偏离

- 不要像另一款手游菜单
- 不要极简到失去叙事气氛
- 不要把所有信息都堆成统一列表卡片

## 5. 角色气质图简报

首批优先角色：

- 阿斯特拉：锋锐、压线、年轻但背负决断
- 莉奥拉：圣辉、克制、温柔但不脆弱
- 凯尔：厚重、沉稳、护线感强
- 伊芙琳：轻锐、机警、带回廊生存感
- 赛琳：组织感、旗手感、让人联想到共鸣与编队

要求：

- 角色立绘、头像、动作表和技能气质应互相一致
- 一眼能分出职业语义和战场职责
- 不要只是换发色的同模版幻想角色
